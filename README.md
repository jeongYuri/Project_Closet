# ☀️👖Project_Closet / ## 내 손의 옷장

### 소개

### 목적
-날씨에 적합한 옷 종류 추천을 통하여 사용자들의 외출 준비시간을 감소시키고자 한다.                                                                                 
-어플리케이션을 이용하여 나만의 옷장 기능을 통해 내 옷을 저장하고, 카테고리 별 옷을 설정하여 관리한다.
